The app is in WIP state, which means to run the app you need to:
	-have python 3.8+ installed
	-install dependencies from requirements.txt file
	-install ffmpeg and add it to the system PATH
	-run app.py from your terminal/IDE
	-enter 127.0.0.1:5000 in your web browser
	-Now you have access to the app. Enjoy the WIP state!

At the moment, the most up to date branch is Text-burning-with-ffmpeg
What you see on the app MAY VARY from the final render, because I'm trying to translate CSS view into ffmpeg's ASS style. Tried my best, but there is difference between rendering engines at the front-end and the back-end. Still, scaling applied by backend rendering is the same on the frontend, so it should be as close as possible.
Also - front-end is mostly made with AI. I did briefly optimize it and changed some parts to my liking, but I'll work on it later on. For now I want it to get the job done, because my main focus with this project is the text generation & customization, and work on my own vocal removing neural network.

App is a PC only for now. May make android version in future.
